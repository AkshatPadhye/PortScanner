# Flask Port Scanner

## Description
A professional cybersecurity port scanner built using Python and Flask. The application scans common ports on a target machine and displays whether the ports are open or closed using a modern web interface.

## Technologies Used
- Python
- Flask
- Socket Programming
- HTML
- CSS

## Features
- Scan common ports
- Professional UI dashboard
- Real-time scan results
- Beginner-friendly cybersecurity project
